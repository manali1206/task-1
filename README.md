# 🌐 Personal Portfolio Website

A responsive multi-page portfolio website built using HTML5, CSS3, and JavaScript.

## 📌 Project Overview

This project is a personal portfolio website developed as part of a web development assignment. It showcases personal information, skills, projects, and contact details through a clean and user-friendly interface.

## ✨ Features

- Multi-page website
- Responsive design
- Semantic HTML5 structure
- Navigation menu
- About Me section
- Projects showcase
- Contact form
- External CSS styling
- JavaScript form handling

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run the Project

1. Download or clone the repository.

```bash
git clone https://github.com/your-username/portfolio.git
```

2. Open the project folder in VS Code.

3. Install the Live Server extension.

4. Right-click on `index.html`.

5. Select **Open with Live Server**.
